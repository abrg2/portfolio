# Personal Portfolio Website

## Overview

This project is a personal portfolio website developed as part of a Frontend course. The purpose of the website is to present my work, technical skills, and areas of interest in a clear and professional way.

The portfolio highlights both professional projects and personal tooling, with a focus on practical systems, web applications, and low-level software development.

## Features

* Single-page responsive design
* Clean and minimal UI with a dark theme
* Project showcase with images and descriptions
* Dedicated section for tooling and hobby projects
* Deployment workflow visualization
* Structured presentation of technical skills

## Technologies Used

* HTML5
* CSS3 (Grid, Flexbox)
* JavaScript (basic interactivity)

## Project Structure

```
/portfolio
│── index.html
│── style.css
│── /images
```

## Sections

* **Hero** – Introduction and role
* **About** – Background and focus areas
* **Projects** – Professional and team-based work
* **Tooling & Hobby Projects** – Specialized tools and low-level work
* **Engineering & Deployment** – Deployment workflow and system design
* **Skills** – Technical overview
* **Contact** – Links and contact information

## Deployment

The project is deployed using GitHub Pages.

To run locally:

1. Clone the repository
2. Open `index.html` in a browser

## Development Process

The development process focused on:

* Planning content and structure before implementation
* Iteratively improving layout and design
* Refining project descriptions for clarity and professionalism
* Keeping the implementation simple and maintainable
